# 网址百宝库

一个纯前端的网址聚合页面，支持分类浏览和关键词搜索功能。

## 🌟 功能特性

### 核心功能
- **分类导航**：包含"全部"、"我的"及多维度业务分类（游戏、工具、社交、办公、学习、娱乐、资讯）
- **网址展示**：图标+名称的简洁卡片式展示，点击直接跳转
- **关键词搜索**：实时搜索网址名称和URL，支持清空和回车触发
- **数据筛选**：分类筛选和搜索筛选可叠加使用

### 交互优化
- **响应式设计**：适配PC端和移动端
- **排序功能**：支持默认排序、按名称排序、按分类排序
- **视觉反馈**：悬停效果、激活状态、加载动画
- **我的网址**：特殊标识和配色，突出显示自研网址

### 技术特点
- **纯前端实现**：无需后端接口和数据库
- **Vue 3 + Vite**：现代化开发体验
- **组件化架构**：模块化设计，易于维护和扩展
- **性能优化**：计算属性缓存，流畅的用户体验

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发环境
```bash
npm run dev
```
访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

## 📁 项目结构

```
src/
├── components/          # Vue组件
│   ├── CategoryNav.vue    # 分类导航组件
│   ├── WebsiteGrid.vue    # 网址网格展示组件
│   ├── SearchBox.vue      # 搜索框组件
│   └── SortControls.vue   # 排序控制组件
├── data/
│   └── websites.js        # 网址数据和分类定义
├── App.vue              # 主应用组件
└── main.js              # 应用入口文件
```
